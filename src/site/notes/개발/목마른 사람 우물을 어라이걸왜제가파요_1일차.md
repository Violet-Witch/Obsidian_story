## [0. 시작 전에]
### python🐍
- 버전은 3.11.5입니다.
- 개발자들이 최신 버전 안 쓰는데는 안정화 부족 등의 이유가 있습니다.
- 물론 그런거 무시하고 최신 버전에 나온 모든 기능을 써보겠다는 개발자도 있습니다.
- 전 귀찮아서 최신 버전 깔아쓰는 개발자인데 개발 설정이 꼬여서 이렇게 됐습니다.
- 아무 버전이나 쓰셔도 괜찮습니다. 어차피 안정화 이런건 우리 얘기 아닙니다.

### 왜 Django 였는가❓
- 뭔가 합리적인 이유가 있을거라고 기대하시면 안 됩니다. 그냥 할 줄 아는게 얘밖에 없습니다.
- 라이트한 웹사이트를 만들고 싶으시다면 **Flask**가 휠씬 낫습니다. 그거 배우세요.
- Django는 이 짓 하기에 상당히 무겁습니다. 다시 한 번 말하지만 전 Flask를 할 줄 몰라서 선택지가 없었을 뿐입니다.
- [플라스크 기초 배우러가기](https://han-py.tistory.com/339) 좋은 분이 정리를 해주신 게 있습니다.

### VSCODE 설정 💻
- Visual Studio **Code**입니다. 헷갈려서 Visual Studio 까시면 대참사 납니다.
- 프로그래머용 텍스트 에디터입니다. 제가 써봤던 것 중에 제일 편하고 좋습니다.
- 참고로 제가 쓰던 Atom은 죽었습니다. **진짜로.** 업데이트 지원 종료됨. 나도 방법이 없었어.
- 뭘 깔아야 하는가?(확장자)

	2. Pylance
	3. Python Debugger
